# Cafe Sales Analysis (Excel) 📊

## Project Goal
Analyzed 500+ cafe transactions to find revenue trends and customer preferences.

## What I Did:
* **Cleaned Data:** Fixed messy entries in the raw dataset.
* **Analysis:** Used Pivot Tables to group sales by category and payment method.
* **Visualization:** Created a dynamic dashboard with slicers for easy filtering.
